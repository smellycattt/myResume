    Key Generation in A5/1 Encryption Algorithm using Versatile LFSR Circuits

    IEEE ICSPVCE 2019,DTU

    In this project we have worked on randomizing the key generation process for the A5/1 Algorithm of Encryption. The Key generation is done with the help of a LFSR by providing a 7 Bit input where 4 Bits are the initial seed of the LFSR that is used as a pseudo-random number generator and 3 bits are used for the initialization of the counter. The modified LFSR circuit is then used to generate the key and that key is used for encryption/decryption of the data through the A5/1 Algorithm.
